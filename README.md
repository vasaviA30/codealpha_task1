# Simple E-Commerce Store

A basic *E-Commerce Web Application* developed using *HTML, CSS, JavaScript* for the frontend and *Django / Express.js* for the backend.

The project allows users to browse products, add items to cart, register/login, and place orders.

---

# Features

- User Registration & Login
- Product Listings
- Product Details Page
- Shopping Cart System
- Order Processing
- Responsive User Interface
- Database Storage for:
  - Users
  - Products
  - Orders

---

bash

bash
